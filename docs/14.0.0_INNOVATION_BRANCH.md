# DNSForge 14.0.0 Innovation Branch

DNSForge 14.0.0 opens the innovation branch after the 13.0.x production maintenance freeze.

## Branch responsibilities

- `13.0.x`: maintenance only, security/stability/compatibility fixes.
- `14.0.x`: new product evolution, Manager, DNSBeat and DNSSync enhancements.

## Non-regression rule

DNSForge Agent remains the only component allowed to modify BIND local configuration and data.

DNSForge Manager, DNSBeat and DNSSync orchestrate, monitor and synchronize through DNSForge Agent boundaries.

## Initial 14.x focus

- Manager UI/API evolution
- Multi-site inventory
- DNSSync orchestration hardening
- DNSBeat monitoring and alerting
- PostgreSQL backend implementation
- RBAC and audit hardening

## Compatibility baseline

The 14.0.0 branch starts from the validated 13.0.x GA baseline and must keep all existing CLI commands available locally on each installed DNSForge server.
