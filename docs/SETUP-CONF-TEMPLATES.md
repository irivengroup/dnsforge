# Modèles setup.conf

Les modèles `install/templates/*/setup.conf` intègrent désormais les variables présentes dans `src/settings`.

Profils :

```text
install/templates/authoritative/setup.conf
install/templates/proxy-forwarder/setup.conf
install/templates/proxy-hybrid/setup.conf
```

La couverture est listée dans :

```text
install/templates/VARIABLE-COVERAGE.md
```
